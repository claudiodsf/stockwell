stockwell
=========

Time-frequency analysis through Stockwell transform.

:Release: |release|
:Date:    |release date|
:Project Page: `GitHub`_

Description
-----------

Stockwell provides a fast implementation of the Stockwell transform
(S-transform), its inverse, and related utilities like the Hilbert
transform and sine tapers.

It uses C extensions (via FFTW3) for high performance.
